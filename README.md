# SquamateGenomics

This repository contains scripts used in the Springer Book "Statistical Population Genomics 2nd Edition", edited by J. Dutheil, chapter "Population and Evolutionary Genomics of Lizards and Snakes".


Data for Squamates was downloaded from NCBI as shown in [20-04-2025.SquamateGenomes.collectdata.Rmd](scripts/20-04-2025.SquamateGenomes.collectdata.Rmd).

Quality of each genome assembly was assessed with BUSCO. The script is available at [busco.sh](scripts/busco.sh).

Parts of Figure1 were created with [22-05-2025.SquamateGenomesFigure1.Rmd](scripts/22-05-2025.SquamateGenomesFigure1.Rmd) and [16-06-2025.MammaliaAvesAmphibiaGenomes.Rmd](scripts/16-06-2025.MammaliaAvesAmphibiaGenomes.Rmd) put together with Affinity Designer. The state of Squamate Genomics was compared to that of Mammals, Birds and Amphibians using the data available in the [data folder](data/Mammalia_Aves_Amphibia).

Figure2 was created with [27-05-2025.SquamateGenomesFigure2.Rmd](scripts/27-05-2025.SquamateGenomesFigure2.Rmd).
